## How to run project

make sure you have installed:

- [Node.js](https://nodejs.org/en/)
- [pnpm](https://pnpm.io/)

then install all dependencies

```bash
pnpm install
```

run project

```bash
pnpm start
```

in development mode

```bash
pnpm start:dev
```

or build

```bash
pnpm start:build
```
